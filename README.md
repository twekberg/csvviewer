# INSTALLATION

## Create A Virtualenv And Install Dependencies

Run these commands:

	cd src/
	git clone git@github.com:twekberg/csvviewer.git
	cd csvviewer
	python -m venv venv

This completes the virtualenv setup.
	
To Complete the installation Run these commands:

	dos2unix venv/Scripts/activate
	source venv/Scripts/activate
	pip install pip -U
	pip install -r requirements.txt

# Sample Command

Here is a command I used to scan ny checking account which is stored
as a CSV file:

	cd CHECK-BECU-CHECKING/
	grep 2988.00 checking.csv | ../src/csvviewer/bin/csvviewer '-d;' -

This looks for one or more checks containing 2988.00.
